Class Widget.FileChooser

This class defines a popup window that have a file chooser dialog.

Checks if the keyboard has the focus in this widget.

Returns:

    True if keyboard has the focus and false otherwise.
Widget.FileChooser:isMouseFocused ()
Checks if the mouse has the focus in this widget.

Returns:

    True if mouse has the focus and false otherwise.
Widget.FileChooser:new (mode, ...)
Creates a new widget object.

Parameters:

  • mode A string with these options: "open" for show a open file dialog, and "save" for show a save file dialog.
  • ... A table with these optional parameters: If the mode is "open", the extra arguments are "onSuccess" and "onCancel" functions. If the mode if "save", the extra arguments add "filename" strig (the default name of file) in the dialog.
Widget.FileChooser:textInput (t)
Adds text to input widgets. Use this function with love.textinput.

Parameters:

  • t The text to add to a input widget.
Widget.FileChooser:update (dt)
Updates the widget.

Parameters:

  • dt The time in seconds since the last update.
