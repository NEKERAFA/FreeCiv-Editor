getFlag (flags, name, pos[, default])
Checks if a flag parameter exists or is valid.

Parameters:

  • flags Table that represents a hashmap with all flags values.
  • name Key of the flag to check.
  • pos The position of the parameter in the function.
  • default If default is not nil, if the flag do not exist, with return this value. (optional)

isPositiveInteger (n, pos[, max])
Checks if the number is positive integer. Is like isNaturalNumber method, but includes the zero number.

Parameters:

  • n The number to check.
  • pos The position of the parameter in the function.
  • max If max is not nil, checks if the number is less or equals that it. (optional)
