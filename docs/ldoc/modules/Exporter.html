export (file, name, rows, cols, terrain, startpos)
Gets the parse table and translate it into a Freeciv map file.

Parameters:

  • file The file where the map is saved.
  • name The name of the map.
  • rows The number of rows in the map.
  • cols The number of columns in the map.
  • terrain A table with the map terrain.
  • startpos A table with the start position of the civilizations.
